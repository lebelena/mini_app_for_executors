# Начало работы

Чтобы начать работу в MiniApp, необходимо получить пригласительную ссылку от контрагента в телеграме. После перехода по ссылке откроется MiniApp с главной страницы приложения.

![_screen_start_03](_images/_screen_login_executor.png){.center width=1200}

Как исполнителю вам доступны разделы «Результат миссий»(2), «Загрузка отчёта»(1) и «Меню»(3). Подробное описание каждого раздела представлено ниже.

## 1. Загрузка отчёта

После выполнения любой задачи требуется загрузить отчёт. 

![_screen_start_03](_images/_screen_report_upload.png){.center width=1200}

Отчёт загружается в систему по штрих-коду на бланке отчётного документа. Если штрих-код не распознан, можно ввести номер документа вручную.

{% note warning %}

Если вы используете устройство с операционной системой Android, для работы необходимо перейти в web-версию системы. Функционал web-версии полностью аналогичен возможностям MiniApp в телеграме.

{% endnote %}

В зависимости от задачи для отчёта требуются фотографии разных категорий. Каждый тип отчётных снимков выделен в отдельную группу, например: «Фото до начала работ» или «Фото маркировки РМ». Загружайте фотографии строго в соответствии с указанной категорией.

![_screen_start_03](_images/_screen_report_photo.png){.center width=1200}

При возникновении ошибок обращайтесь в техподдержку. 

## 2. Результат миссий

Результат миссий содержит результаты проверок отчётов координаторами.

![_screen_start_03](_images/_screen_mission_start.png){.center width=1200}

Если отчёт еще находится на проверке у координатора, он будет находиться на вкладке **«Проверяемые результаты».** 

![_screen_start_03](_images/_screen_report_check.png){.center width=1200}

Если отчёт принят, то он будет перемещен в **«Историю миссий»**: там хранятся все принятые отчёты.

![_screen_start_03](_images/_screen_report_story.png){.center width=1200}

Если отчёт отклонён, уведомление об этом приходит в Telegram-чат. В сообщении будет кнопка «Исправить», которая открывает конкретную отклонённую заявку. 

![_screen_start_03](_images/_screen_report_edit.png){.center width=1200}

И в чате, и в MiniApp будет виден комментарий координатора с указанием причины отклонения. Необходимо исправить отчёт в соответствии с замечаниями и повторно отправить его на проверку.

![_screen_start_03](_images/_screen_report_fix.png){.center width=1200}

{% note warning %}

Если отчёт не будет одобрен, оплата по нему не будет произведена. 

{% endnote %}

## 3. Меню

![_screen_start_03](_images/_screen_menu.png){.center width=1200}

В боковом меню доступно шесть вкладок, каждая из которых рассмотрена ниже.

### 3.1. Профиль

Позволяет выйти из профиля в MiniApp.

![_screen_start_03](_images/_screen_menu_profile.png){.center width=1200} 

### 3.2. Дорожная карта

Интерактивная дорожная карта знакомит с историей развития компании и планами по развитию MiniApp. На странице отражены ключевые этапы пути и стратегия на будущее.

![_screen_start_03](_images/_screen_menu_road_map.png){.center width=1200}

### 3.3. Планирование

На вкладке «Основное направление» собраны **задания, на которые координатор назначил вашего контрагента исполнителем,** вкладка «Tobacco» не используется. 

![_screen_start_03](_images/_screen_menu_plan_docs.png){.center width=1200}

Для каждого задания доступно техническое задание (ТЗ), которое едино для всех задач и маршрутов в рамках задания.
 
![_screen_start_03](_images/_screen_menu_plan_tz.png){.center width=1200}

Задачи группируются по маршрутам. Вы можете просмотреть маршрут на встроенной карте. Также для каждой задачи указано контактное лицо — ему можно позвонить прямо из MiniApp. 

![_screen_start_03](_images/_screen_menu_plan_tasks.png){.center width=1200}

### 3.4. Техподдержка

При возникновении ошибок в работе платформы обратитесь в техподдержку.

![_screen_start_03](_images/_screen_tech_sup.png){.center width=1200}

### 3.5. Инструкции

Часто возникающие вопросы собраны на вкладке «Инструкции». 

![_screen_start_03](_images/_screen_menu_instr.png){.center width=1200}

«Правила работы исполнителей» содержат все требования к:
- вам как к исполнителю;
- подготовке до начала выполнения работ;
- проведению работ;
- доставке, передаче и хранении оборудования;
- отчётам, предоставляемым после проведения работ.

В «Основном направлении» описаны часто задаваемые вопросы, связанные с функционалом и логикой работы MiniApp, вкладка «Tobacco» не используется.
