# Экипаж

После входа необходимо добавить исполнителей — членов вашего экипажа. Для этого используйте команду «Собрать экипаж».

![_screen_gant_day](_images/_screen_team_start.png){.center width=1200}

В открывшемся окне выберите контакт, которого хотите пригласить. Исполнитель получит сообщение со ссылкой, при переходе по которой он перейдет в бот. **Исполнитель не будет проходить регистрацию: он автоматически будет закреплён за вами и добавлен в ваш экипаж.**

![_screen_gant_day](_images/_screen_team_add.png){.center width=1200}

{% note warning %}

**Контрагенты и исполнители обладают разными уровня доступа к системе.** Исполнитель видит в меню: профиль, дорожная карта, планирование, техподдержка, инструкции и телеграм-канал. Доступ к финансам и рейтингу у исполнителей отсутствует — они не увидят, кто из команды назначен на какие заказы и какую оплату получают другие члены экипажа.

{% endnote %}

Имя пользователя автоматически подтягивается из его профиля Telegram. Вы можете отредактировать это имя или удалить пользователя из команды при необходимости. 

![_screen_gant_day](_images/_screen_team_edit_del.png){.center width=1200}
